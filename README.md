# RentFlow Admin Dashboard

Platform admin dashboard for managing RentFlow's multi-tenant marketplace.

- `admin.rentflow.com` manages platform-wide tenants, shop owners, and domains.
- `partner.rentflow.com` is the shared back office for each shop owner.
- `{shop}.rentflow.com` is the public rental storefront for each tenant.

## Development

```bash
npm install
npm run dev
```
